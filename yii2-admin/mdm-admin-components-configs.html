Class mdm\admin\components\Configs

All Classes | Properties | Methods
Inheritancemdm\admin\components\Configs » yii\base\Object
Available since version1.0

Configs Used for configure some value. To set config you can use yii\base\Application::$params

return [

    'mdm.admin.configs' => [
        'db' => 'customDb',
        'menuTable' => 'admin_menu',
    ]
];

or use Yii::$container

Yii::$container->set('mdm\admin\components\Configs',[
    'db' => 'customDb',
    'menuTable' => 'admin_menu',
]);
